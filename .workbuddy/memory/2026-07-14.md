## 修复：点击"启动时光机"无反应

- **根因**：前端 JS 用 `fetch('index.json')` 加载事件索引，在预览浏览器的安全上下文中 fetch 被阻塞
  - Promise 既不 resolve 也不 reject → click 事件监听器永远挂不上 → 点击无反应
- **修复**：改为内联 JSON 方案
  - `renderHomepage(index)` / `renderEventPage(event, titleMap, index)` / `renderArchivePage(index)` 接收 index 参数
  - 在 HTML 中嵌入 `<script type="application/json" id="event-index">{...}</script>`
  - JS 用 `JSON.parse(document.getElementById('event-index').textContent)` 读取，零网络请求
  - 同时加了 localStorage 可用性检测（某些环境可能禁用 localStorage）
- 三个页面（首页/Event/Archive）全部去掉 fetch，改为内联数据
